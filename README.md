![screenshot](./src/assets/screenshot.png)

## Get Started

### 1. clone this repository

```bash
git clone https://github.com/rChinnnn/electron-Vue3-streaming-template.git
```

### 2. install packages

```bash
npm i
```

### 3. run

```bash
npm run app:dev
```
